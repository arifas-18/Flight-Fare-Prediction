# Flight-Fare-Prediction
This project predicts flight ticket prices using machine learning regression models.

## Files in this Repository
- Flight_Fare_Prediction.ipynb 
- Flight_Fare_Prediction_Final_Report.pdf 
- Flight_Fare.xlsx 

## Problem Type
Regression (predicting flight ticket price)

## Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- XGBoost Regressor

## Best Model
XGBoost with R² score ≈ 0.857
